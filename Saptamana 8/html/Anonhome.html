App Description

Funcționalități

  • Utilizatorul are 2 optiuni:
    • Dupa crearea unui cont/autentificare: ;
      • poate raspunde la intrebari;
      • poate pune intrebari: pentru fiecare intrebare poate alege tagurile si categoria din care face parte,si poate adauga un context daca este cazul;
      • poate cauta intrebari dupa anumite taguri;
      • poate filtra intrebarile dupa taguri, categorii si data publicarii;
      • poate aprecia sau deprecia anumite intrebari sau raspunsuri;
      • poate primi dupa ce a pus sau a raspuns la un numar de intrebari, un bagde, care este de mai multe tipuri;
      • poate modifica anumite setari ale contului si anume: numele, prenumele, email, parola, username, data de nastere, categoriile si tagurile.
    • Foloseste aplicatia ca utilizator anonimn si:
      • poate raspunde la intrebari;
      • poate pune intrebari: pentru fiecare intrebare poate alege tagurile si categoria din care face parte,si poate adauga un context daca este cazul;;
      • poate cauta intrebari dupa anumite taguri
      • poate filtra intrebarile dupa taguri, categorii si data publicarii;
      • nu poate aprecia sau deprecia anumite intrebari sau raspunsuri, este necesar un cont
      • nu poate modifica setarile contului pentru ca acesta nu are un cont
      • nu poate primi un badge, pentru ca activitatea nu este retinuta nicaieri din cauza lipsei contului de utilizator;
      • isi poate crea cont oricand.
